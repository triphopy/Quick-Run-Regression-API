SONIC Quick Start

1. แตกไฟล์ zip นี้ไปไว้ในโฟลเดอร์ที่ต้องการ
2. เปิดโฟลเดอร์ SONIC
3. ดับเบิลคลิก SONIC.exe
4. รอสักครู่ ระบบจะเปิดหน้าเว็บใน browser อัตโนมัติ
5. ถ้า browser ไม่เปิดเอง ให้เข้า http://localhost:8501

การใช้งานเบื้องต้น

- หน้า Run Test:
  อัปโหลด CSV ของคุณ หรือเลือก sample/template ที่มีให้
- หน้า Dashboard:
  ดูภาพรวมผลการรัน และดาวน์โหลดผลลัพธ์
- หน้า Log:
  ดูรายละเอียดราย API และคัดลอกข้อความไปใช้ต่อใน Jira

ข้อสำคัญ

- ต้องเก็บไฟล์ทั้งหมดในโฟลเดอร์ SONIC ไว้ด้วยกัน
- อย่าย้าย SONIC.exe ออกไปเดี่ยว ๆ
- ถ้า Windows เตือนเรื่องความปลอดภัย ให้กด More info > Run anyway ตามนโยบายเครื่องของคุณ
- ถ้าพอร์ต 8501 ถูกใช้อยู่ แอปอาจเปิดหน้าเดิมที่กำลังรันอยู่

ไฟล์ผลลัพธ์

- แอปจะสร้างข้อมูลการรันไว้ในโฟลเดอร์ data
- สามารถล้าง run เก่าและ temp files ได้จากหน้า Dashboard
